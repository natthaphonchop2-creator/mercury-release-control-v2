# Mercury Finance v0.2.2

Mercury Finance is an MCP and Codex plugin for accounting knowledge, ERP connector catalogs, Thai finance workflows, and local-credential ERP execution.

## Included

- One public hosted MCP with 20 tools and a 254-action FlowAccount/PEAK catalog.
- Local ERP credentials remain in the user's workspace and are not stored by the hosted MCP.
- Read and controlled write execution use connector capabilities, risk policy, confirmation, and local audit evidence.
- Thai accounting Skills cover connector setup, company health, VAT, invoice review, management reporting, Mercury Flows, and FlowAccount journal posting.
- Supabase-backed reviewed knowledge includes connector endpoint evidence and cited accounting context.

## Release Integrity

This release is built from an exact reviewed commit, compared through PublicTreeV1, inspected across hosted and repository surfaces, published through a draft-verified asset flow, and finalized as an immutable GitHub Release.
